Concepts

  • Object Detection: 检测目标的类型和种类
  • Single-Shot Dection: 早期的目标检测工程由两个不同的步骤组成：a region proposal network检测出目标的位置，分类器在proposed regions中检测出目标的类型。但是很耗费算力，不能实时检测。而Single-shot模型将这两个步骤压缩为一个步骤，因此大大提高了检测速度和方便硬件部署。
  • Multiscale Featur Maps: 在图像分类任务中，我们是基于最后的卷积特征图来进行预测的。在目标检测中，中间层次的特征图也是十分有用的，因为它们表示了原始图片的各个尺度。因此，一个固定大小的filter在不同尺度的特征图上能够检测出不同大小的objection。
  • Priors: 定义在特定的特征图上的预先计算好的boxes，它们有特定的尺度。被选出来能够更好地匹配图像的bounding boxes.
  • Multibox: 将预测物体的bounding box作为回归问题，检测到的物体的坐标回归到ground truth坐标。此外，对于不同的类别都会根据box产生相应的分数。Priors是一个很好的起点，因为它们是根据ground truths构建的。因此，将会有和priors一样多的先验框，它们大多数将不会包含objection
  • Hard Negative Mining: 只挖掘模型很难识别的负样本。因为很多先验框并不包含objection,这样做有利于正负样本的均衡。

  • Non-Maximum Suppression:检测同一个物体，将会有许多框重叠在一起。NMS就是通过抑制除了最大分数意外的所有预测来移除掉冗余的。

    Some definitions

    Boundary coordinates

    最显然的方式是通过像素坐标x,y来表示objection的边界 但是如果我们不知道图像的实际尺寸的话这就行不通了。所以更好地方法是通过小数形式来表示。这样坐标的尺寸是不变化的，在所有图像上的boxes有同样的测量尺度。

    Center-Size coordinates

    能够表现出box的坐标和尺度

    Jaccard Index

    算出两个box的重叠程度

    Multibox

  • 可能包含或者不包含对象box的坐标。这是一个回归任务

  • box中各种对象类型的得分，这是一个分类任务

    Single Shot Detector(SSD)

    SSD是一个纯粹的CNN，主要由三部分组成：

  • Base convolutions来自已有的图像分类框架，提供初步的特征图

  • Auxiliary convolutions添加到base network之上，提供更高层次的特征图

  • Prediction convulitions定位和识别objection

Base Convolutions – part 1

采用VGG16做基础模型，首先VGG16是在ILSVRC CLS-LOC数据集预训练.然后做出相应的改变来适应本任务。 1. 输入图像尺寸为300, 300 2. 将池化层pool5由原来的stride=2的2X2变成stride=1的3x3,这样特征图尺寸不再减半 3. 不再使用全连接层，抛弃fc8,将fc6和fc7转变为卷积层conv6和conv7

FC-> Convolutional Layer

首先不能对特征图直接地进行全连接操作，需要先将特征图拉伸成1D。灰色的向量表示的是全连接层的参数。 如果用卷积层产生这两个输出: 所以可以看到这两个结果是相同的。那么一个H,W,I的图像，全连接操作输出为N等同于和同样大小，输出为N的kernel做卷积操作

A detour

我们预测物体的位置和类别，但是是以什么形式表现出来的呢？因此我们先来简绍priors

Priors

先验框根据数据集中的ground truth的大小和形状是提前设置好，预测框则以先验框为基准不断调节变换。先验框得设置，作者指出来一下几点： - 先验框被应用conv4_3, conv7, conv8_2, conv9_2, conv10_2, conv11_2,高低层次的feature maps - 如果先验框的尺度是s, 那它的面积等于以s为边的正方形。所有feature map都有一个比列为1:1的extra prior，其尺度为当前尺度和下一个feature map尺度的几何平均值。都有长宽比为1:1, 2:1, 1:2的先验框。更大的feature map有更小的尺度，一次更加适合检测更小的物体 - feature map的每个位置都会有不同长宽比的先验框。

Prediction convulutions

对于在特征图每个位置的先验框，我们想得到： - bounding box的四个偏移量值 - bounding box的n_classed分数， n_classed是物体的总共类别，背景这个类别也被包含在其中

所以我们为这些特征图设置了两个卷积层： - 位置预测卷积层，核为3x3，padding和stride为1,4个filters - 类预测卷积层，核为3x3,padding和stride为1，n_classfilters
